With this tool, for any  given scenario, parcellation / fragmentation can be traced where the output file as a point will contain the attributed from both file and will allow comparisons to be made. 

INPUTS: 
	1. Recent year polygon shapefile 
	2. Past year   polygon shapefile 
OUTPUT: 
	Point Files of new parcels and attributes from two files.
